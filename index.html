I'm a Laravel developer based in Dohuk, graduated from the IT Department at Akre Technical Collage, Duhok Polytechnic University. I build practical, production-ready web applications — from real-time admin dashboards to public-facing systems people actually rely on.

My day-to-day stack is Laravel, Livewire, Bootstrap 5, and MySQL. I care about clean, maintainable code over clever shortcuts, and I like solving the unglamorous problems — deployment quirks, file storage, server configs — that decide whether an app actually works in production, not just on localhost.

Right now I'm focused on NativePHP, turning the Laravel apps I already know how to build into native desktop and mobile experiences without switching ecosystems.
